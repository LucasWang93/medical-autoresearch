# Multi-task learning autonomous loop plan

Goal: jointly optimize 4 MIMIC-IV tasks (mortality, readmission, LOS, phenotyping)
via shared encoder + per-task heads. 7-hour autonomous exploration budget.

Tasks: mimic4_mortality, mimic4_readmission, mimic4_los, mimic4_phenotyping
(drugrec excluded — uses drugs_hist feature key incompatible with shared encoder)
